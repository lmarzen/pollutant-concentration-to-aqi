# pollutant-concentration-to-aqi
A C library that converts pollutant concentrations to Air Quality Index(AQI).

Supports 10 common AQI scales.
- Australia AQI
- Canada AQHI
- Europe CAQI
- Hong Kong AQHI
- India AQI
- Mainland China AQI
- Singapore PSI
- South Korea CAI
- United Kingdom DAQI
- United States AQI

See aqi.h for more information about function usage.
